# Yaroslav Stopenchuk — Portfolio

Personal portfolio website showcasing my work as a Frontend/Full-Stack Developer.

**Live:** [humblezxc.github.io/portfolio](https://humblezxc.github.io/portfolio)

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Deployment:** GitHub Pages

## Features

- Responsive design with glassmorphism UI
- Dark/Light theme support
- Multi-language support (EN, PL, UK)
- Optimized performance (Lighthouse 100/100 SEO)

## Getting Started

```bash
# Install dependencies
yarn

# Start development server
yarn dev

# Build for production
yarn build
```

## Project Structure

```
src/
├── app/           # Next.js App Router pages
├── components/    # React components (Hero, Skills, Projects, Experience, About, Contact)
├── lib/           # Data files and utilities
└── locales/       # i18n translation files (en, pl, uk)
```
